# 🧠 Brain Age Prediction Using MRI

This project predicts the **biological age of the human brain** using MRI scan images and deep learning techniques.
The system analyzes brain MRI images and estimates the brain's age, which can help identify abnormal brain aging patterns and support neurological research.

---

## 🚀 Features

* Upload MRI brain scan images
* Deep learning model for brain age prediction
* Image preprocessing and analysis
* Web interface for predictions
* Database storage for results
* Sample MRI inputs for testing

---

## 🧰 Technologies Used

* **Python**
* **Flask** (Web Framework)

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/yourusername/brain-age-prediction.git

2. Navigate to the project folder

cd brain-age-prediction

3. Install required libraries

pip install -r requirements.txt

---

## ▶️ Run the Application

Run the Flask application:

python app.py

Then open your browser and go to:

http://127.0.0.1:5000

---

## 🧪 How It Works

1. User uploads an MRI brain scan image.
2. The system preprocesses the image.
3. The trained deep learning model analyzes the scan.
4. The model predicts the **estimated brain age**.
5. Results are displayed on the web interface.
